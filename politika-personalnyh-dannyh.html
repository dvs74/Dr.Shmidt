Политика обработки персональных данных

Настоящая Политика обработки персональных данных составлена в соответствии с Федеральным законом №152‑ФЗ и регулирует порядок обработки персональных данных на сайте https://drshmidt.ru.

1. Общие положения

Оператором персональных данных является владелец сайта https://drshmidt.ru. Контактный email: info@drshmidt.ru.

Оператор обеспечивает соблюдение прав и свобод человека и гражданина при обработке персональных данных.

2. Персональные данные, которые обрабатываются

  • Имя
  • Номер телефона
  • Адрес электронной почты
  • Иные данные, переданные пользователем через форму обратной связи

3. Цели обработки персональных данных

  • Обратная связь с пользователем
  • Обработка обращений
  • Предоставление консультаций

4. Права пользователя

  • Получать информацию об обработке своих данных
  • Требовать уточнения или удаления данных
  • Отзывать согласие на обработку данных

5. Контакты

По вопросам обработки персональных данных вы можете обратиться по адресу: info@drshmidt.ru.
